Pattern of Questions

There are 7 different categories of questions which can be asked

  1. MCQ - I :In this type of question ONLY ONE OPTION IS CORRECT
  2. MCQ - II :In this type of question MORE THAN ONE OPTION can be correct with NO PARTIAL MARKING
  3. MCQ - III :In this type of question MORE THAN ONE OPTION can be correct with PARTIAL MARKING
  4. INTEGER TYPE :In this type of question, answer will be in the form of INTEGRAL VALUES ROUNDED UPTO NEAREST INTEGER
  5. FITB :In this type of question, answers have to be FILLED IN THE BLANKS with proper spelling.
  6. True/False :In this type of question, ask if the given statement is TRUE or FALSE
  7. Subjective :In this type of question ,answer is expected to be submitted in the form of TEXT or PDF. Maximum word limit is 400 words.
