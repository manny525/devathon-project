    Pattern of Questions

    There are 7 different categories of questions which can be asked

    1. MCQ - I : This type of question will be in MCQ - I color. In this type of question ONLY ONE OPTION IS CORRECT
    2. MCQ - II : This type of question will be in MCQ - II color. In this type of question MORE THAN ONE OPTION can be correct with NO PARTIAL MARKING
    3. MCQ - III : This type of question will be in MCQ - III color. In this type of question MORE THAN ONE OPTION can be correct with PARTIAL MARKING
    4. INTEGER TYPE : This type of question will be in Integer Type color. In this type of question, answer will be in the form of INTEGRAL VALUES ROUNDED UPTO NEAREST INTEGER
    5. FITB : This type of question will be in Fill in the blanks color. In this type of question, you have to FILL IN THE BLANKS with proper spelling.
    6. True/False : This type of question will be in True/False color. In this type of question you have to select if the given statement is TRUE or FALSE
    7. Subjective : This type of question will be in Subjective color. In this type of question you have to submit the written answer in form of TEXT or PDF. Maximum word limit is 400 words.
